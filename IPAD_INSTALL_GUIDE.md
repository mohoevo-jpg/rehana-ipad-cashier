# دليل تثبيت تطبيق الكاشير على الآيباد

هناك طريقتان لتثبيت وتشغيل التطبيق على الآيباد:

## الطريقة الأولى: تطبيق ويب (PWA) - الأسرع والأسهل
هذه الطريقة لا تتطلب جهاز Mac وتعمل فوراً.

1. تأكد من تشغيل السيرفر على جهاز الكمبيوتر (الأمر `npm run dev` يعمل حالياً على المنفذ 3003).
2. افتح متصفح **Safari** على الآيباد.
3. أدخل عنوان IP الخاص بجهاز الكمبيوتر متبوعاً بالمنفذ 3003.
   - مثال: `http://192.168.1.X:3003`
   - (يمكنك معرفة عنوان IP من خلال إعدادات الشبكة في الكمبيوتر).
4. بعد تحميل الصفحة، اضغط على زر **المشاركة (Share)** (أيقونة مربع وسهم للأعلى).
5. اختر **إضافة إلى الشاشة الرئيسية (Add to Home Screen)**.
6. اضغط **إضافة (Add)**.

سيظهر التطبيق كأيقونة مستقلة على شاشة الآيباد، وسيعمل بملء الشاشة وبدون شريط المتصفح، كما يدعم العمل بدون إنترنت (Offline).

---

## الطريقة الثانية: التثبيت عبر Sideloadly (Windows/Mac)
هذه الطريقة تسمح لك بتثبيت التطبيق كتطبيق أصلي باستخدام جهاز الكمبيوتر فقط.

1. **تحميل ملف التطبيق (IPA):**
   - اذهب إلى صفحة المشروع على GitHub.
   - اضغط على تبويب **Actions** في الأعلى.
   - اضغط على آخر عملية بناء (Build iOS IPA).
   - في أسفل الصفحة، ستجد ملفاً باسم `cashier-app-ipa`. قم بتحميله وفك الضغط عنه لتحصل على ملف `App.ipa`.

2. **التثبيت:**
   - قم بتوصيل الآيباد بالكمبيوتر.
   - افتح برنامج **Sideloadly**.
   - اسحب ملف `App.ipa` إلى داخل البرنامج.
   - أدخل حساب Apple ID الخاص بك واضغط **Start**.
   - بعد الانتهاء، سيظهر التطبيق على الآيباد.
   - اذهب إلى **الإعدادات > عام > VPN وإدارة الجهاز** ووافق على الشهادة لتشغيل التطبيق.

## الطريقة الثالثة: البناء اليدوي (لمستخدمي Mac فقط)
هذه الطريقة تتطلب وجود جهاز **Mac** وبرنامج **Xcode**.

### الخطوات (على جهاز Mac):
1. قم بتحميل المشروع من GitHub:
   ```bash
   git clone https://github.com/mohoevo-jpg/rehana-ipad-cashier.git
   cd rehana-ipad-cashier
   ```
2. افتح "الطرفية" (Terminal) داخل المجلد.
3. نفذ الأوامر التالية لتجهيز المشروع:
   ```bash
   npm install
   npx cap sync
   ```
4. لفتح المشروع في Xcode:
   ```bash
   npx cap open ios
   ```
5. في برنامج Xcode:
   - قم بتوصيل الآيباد بالـ Mac.
   - اختر جهاز الآيباد من قائمة الأجهزة في الأعلى.
   - اضغط على زر التشغيل (Run) لتثبيت التطبيق على الآيباد.

### ملاحظة لمستخدمي Windows:
لا يمكن إنشاء ملفات تطبيقات iOS (.ipa) مباشرة من نظام Windows بسبب قيود شركة Apple. يجب استخدام جهاز Mac للمرحلة النهائية من التصدير.
